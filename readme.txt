/***** Instruction for Frontend *****/

1. Go to react-posts folder , npm install 
2. In frontend folder same open two terminal and in one run react , in another run fake json server


/***** Instruction for server(Backend) *****/

1. Go to server folder , npm install 
2. Set up your google cloud console and paste their key in dotenv file 
3. Run the server npm start
